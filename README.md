# geoguessr-cheat#

Ferramenta simples que tira print da tela, envia pro **Gemini Flash** e tenta adivinhar em qual país/estado/cidade você está no GeoGuessr.

### Como usar

1. Instale as dependências:
```bash
pip install customtkinter mss pillow requests

Coloque sua API Key do Google Gemini no código
(linha: API_KEY = "sua-chave-aqui")

Execute:

python geo.py
